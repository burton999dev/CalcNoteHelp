# CalcNote Pro

### 1. Les publicités disparaissent totalement
Toutes les publicités de l’application sont masquées pour une expérience plus fluide.

- La publicité plein écran au démarrage (toutes les 6 heures) est supprimée, vous pouvez calculer dès l’ouverture.
- Les bannières en haut des écrans disparaissent, libérant plus d’espace d’affichage.
- Les publicités natives dans le menu latéral et dans les réglages sont aussi masquées pour accéder plus vite aux fonctions voulues.

### 2. Gérez librement l’historique des fichiers
Traitez vos fichiers en cours avec davantage de sérénité.
<sub><a href="https://www.youtube.com/watch?v=M9TqYfH1bqo" target="_blank"><img src="https://raw.githubusercontent.com/burton999dev/CalcNoteHelp/refs/heads/master/images/all/yt_icon_red_digital.png" width="32" height="32" style="vertical-align: text-bottom; margin-left: 6px;"></a></sub>

- Chaque enregistrement est sauvegardé automatiquement dans l’historique. (**Même dans l’offre gratuite l’historique est conservé, vous accédez donc immédiatement aux versions passées après le passage à Pro.**)
- Depuis le Gestionnaire de fichiers, affichez la liste des versions et restaurez le point souhaité.
- Restaurez facilement un fichier supprimé par erreur.

### 3. Widget flottant disponible
Un widget pratique pour calculer et prendre des notes tout en laissant les autres apps ouvertes.
<sub><a href="https://www.youtube.com/watch?v=paxUzu6x7sE" target="_blank"><img src="https://raw.githubusercontent.com/burton999dev/CalcNoteHelp/refs/heads/master/images/all/yt_icon_red_digital.png" width="32" height="32" style="vertical-align: text-bottom; margin-left: 6px;"></a></sub>

- Il s’affiche en surimpression semi-transparente pour consulter des résultats ou noter sans interrompre votre tâche.
- Lancez-le à tout moment en un geste depuis la barre d’état.
- Par exemple, notez le prix vu dans le navigateur sans changer d’application.

### 4. Créez un nombre illimité de listes définies par l'utilisateur
Vous pouvez créer autant de **listes définies par l'utilisateur** que vous voulez pour regrouper vos valeurs et unités fréquentes.
<sub><a href="https://www.youtube.com/watch?v=2UZjLNSWFs0" target="_blank"><img src="https://raw.githubusercontent.com/burton999dev/CalcNoteHelp/refs/heads/master/images/all/yt_icon_red_digital.png" width="32" height="32" style="vertical-align: text-bottom; margin-left: 6px;"></a></sub>

- L’offre gratuite se limite à une seule liste définie par l'utilisateur ; Pro permet d’en créer sans limite.
- Alternez les listes selon l’usage et boostez fortement votre efficacité.
