## 1. Вступ
Ця Політика конфіденційності описує підхід CalcNote, калькуляторного застосунку, до обробки персональних даних. Ми наголошуємо, що CalcNote сам по собі не збирає персональну інформацію або дані про місцезнаходження користувачів. Однак важливо зазначити, що бібліотеки третіх сторін, використані в застосунку, можуть збирати певну інформацію. Ця політика має на меті прозоро пояснити, як обробляються дані та з якою метою вони можуть використовуватися.

## 2. Збір і використання персональної інформації
CalcNote створено з пріоритетом на захист приватності користувачів. Тому сам застосунок не збирає персональну інформацію, зокрема імена, адреси електронної пошти або дані про місцезнаходження. Водночас CalcNote використовує такі бібліотеки третіх сторін, які можуть збирати інформацію:

- Admob: Використовується для показу реклами. Зібрані дані можуть включати взаємодію користувача з рекламою та анонімні ідентифікатори.
- Firebase Analytics: Допомагає зрозуміти поведінку користувачів у застосунку, збираючи анонімні дані про використання та взаємодії.
- Firebase Crashlytics: Використовується для моніторингу продуктивності застосунку та виявлення збоїв. Збирає дані про збої та системну інформацію, важливі для підвищення стабільності застосунку.

Кожна з цих служб працює відповідно до власної політики конфіденційності та може обробляти дані згідно зі своїми правилами. Користувачам рекомендується переглянути відповідні політики конфіденційності для отримання детальнішої інформації. Важливо уточнити, що хоча ці служби збирають дані для конкретних цілей, таких як реклама, аналітика та продуктивність застосунку, CalcNote сам по собі не збирає персональні дані користувачів.

## 3. Обробка даних третіми сторонами
CalcNote використовує декілька сторонніх сервісів для покращення користувацького досвіду та функціональності застосунку. Ці сервіси, зокрема Admob, Firebase Analytics і Firebase Crashlytics, регулюються власними політиками конфіденційності та умовами обробки даних. Вони обробляють дані користувачів відповідно до Загального регламенту про захист даних (GDPR) та інших застосовних законів.

- Admob: Керує рекламними даними та забезпечує показ користувачам таргетованої й релевантної реклами. Докладніше див. [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html).
- Firebase Analytics і Firebase Crashlytics: Використовуються для розуміння взаємодії користувачів у застосунку та підтримання його стабільності. Детальна інформація наведена у [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms).

Рекомендуємо користувачам ознайомитися з цими документами, щоб зрозуміти, як їхні дані обробляються та захищаються цими сторонніми сервісами.

## 4. Права користувачів
Відповідно до GDPR користувачі мають певні права щодо своїх персональних даних. До них належать:

- Право на доступ: Користувачі можуть запитати інформацію про те, як використовуються та обробляються їхні дані.
- Право на виправлення: Користувачі мають право виправити неточні або неповні дані.
- Право на видалення: Також відоме як «право бути забутим»; дозволяє запитувати видалення даних за певних обставин.
- Право на обмеження обробки: Користувачі можуть вимагати припинення обробки своїх даних.
- Право на переносимість даних: Дозволяє користувачам отримати та повторно використовувати свої дані для власних цілей у різних сервісах.
- Право на заперечення: Користувачі можуть заперечувати проти обробки своїх персональних даних у певних випадках, наприклад для цілей прямого маркетингу.

CalcNote заохочує користувачів користуватися цими правами за потреби. Для запитів або питань щодо персональних даних користувачі можуть звертатися безпосередньо до відповідного стороннього постачальника послуг, оскільки CalcNote самостійно не обробляє персональні дані.

## 5. Захист даних
Безпека та захист даних користувачів є пріоритетом для CalcNote. Хоча застосунок не збирає персональні дані, ми забезпечуємо, щоб інтегровані сторонні сервіси дотримувалися суворих стандартів захисту даних. Ці сервіси повинні впроваджувати відповідні заходи безпеки для захисту даних користувачів від несанкціонованого доступу, зміни, розкриття або знищення. У малоймовірному випадку порушення безпеки даних відповідний сторонній сервіс вживе необхідних заходів для мінімізації можливих збитків і повідомить постраждалих користувачів, як того вимагає закон.

Зміни будуть повідомлені через застосунок або офіційний вебсайт, що дозволить користувачам ознайомитися з оновленими умовами. Оновлена політика набирає чинності з дати публікації. Ми рекомендуємо користувачам періодично переглядати політику конфіденційності, щоб бути поінформованими про те, як захищаються їхні дані.
