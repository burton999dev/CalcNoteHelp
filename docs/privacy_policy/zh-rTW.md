## 1. 前言
本隱私權政策說明 CalcNote（計算機應用程式）在處理個人資料方面的作法。我們強調 CalcNote 本身不會收集使用者的任何個人資訊或位置資料。然而，需注意的是，應用程式中使用的第三方程式庫可能會收集某些資訊。本政策旨在透明地說明資料如何被處理，以及可能用於哪些目的。

## 2. 個人資訊的蒐集與使用
CalcNote 以使用者隱私為優先設計。因此，應用程式本身不會收集任何個人資訊，包括但不限於姓名、電子郵件地址或位置資料。不過，CalcNote 整合了以下第三方程式庫，可能會收集資訊：

- Admob：用於投放廣告。所收集的資料可能包含使用者與廣告的互動以及匿名識別碼。
- Firebase Analytics：透過蒐集有關使用與互動的匿名資料，協助理解使用者在應用程式中的行為。
- Firebase Crashlytics：用於監控應用程式效能並識別當機。會收集與當機相關的資料與系統資訊，對提升應用程式穩定性至關重要。

上述各服務皆依其自身的隱私權政策運作，並可能依其指引處理資料。建議使用者查閱各服務的隱私權政策以取得更多資訊。必須明確指出，雖然這些服務會為廣告、分析與應用程式效能等特定目的收集資料，但 CalcNote 本身不會獨立收集使用者的個人資料。

## 3. 第三方資料處理
CalcNote 使用多項第三方服務以提升使用者體驗與應用程式功能。這些服務（包含 Admob、Firebase Analytics 與 Firebase Crashlytics）皆受其各自的隱私權政策與資料處理條款規範，並依據 GDPR（一般資料保護規範）及其他適用法律處理使用者資料。

- Admob：管理廣告資料，確保向使用者顯示具針對性且相關的廣告。詳情請參閱 [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html)。
- Firebase Analytics 與 Firebase Crashlytics：用於理解應用程式內的使用者互動並維持應用程式穩定性。詳細資訊請參閱 [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms)。

我們建議使用者檢視上述文件，以了解第三方服務如何處理並保護其資料。

## 4. 使用者權利
依據 GDPR，使用者對其個人資料享有特定權利，包括：

- 存取權：使用者可要求了解其資料如何被使用與處理。
- 更正權：使用者有權更正不正確或不完整的資料。
- 刪除權：亦稱「被遺忘權」，允許使用者在特定情況下要求刪除資料。
- 限制處理權：使用者可要求停止處理其資料。
- 資料可攜權：允許使用者取得其資料並在不同服務間為自身目的重複使用。
- 反對權：使用者可在特定情況下（例如直接行銷目的）反對處理其個人資料。

CalcNote 鼓勵使用者在需要時行使上述權利。由於 CalcNote 本身不處理個人資料，關於個人資料的請求或詢問，使用者可直接聯絡相關第三方服務提供者。

## 5. 資料保護
使用者資料的安全與保護是 CalcNote 的優先事項。雖然應用程式本身不收集個人資料，但我們確保整合的第三方服務遵循嚴格的資料保護標準。這些服務必須實施適當的安全措施，以防止未經授權的存取、變更、揭露或毀損。若發生資料外洩，相關第三方服務將採取必要措施以減輕潛在損害，並依法律要求通知受影響的使用者。

## 6. 隱私權政策的變更
CalcNote 保留隨時更新或修改本隱私權政策的權利。變更將透過應用程式或官方網站通知，以便使用者查閱更新後的條款。修訂後的政策自發布日起生效。我們建議使用者定期查閱隱私權政策，以了解其資料如何受到保護。在隱私權政策變更後繼續使用 CalcNote，即表示接受這些變更。

## 7. 聯絡資訊
若對本隱私權政策或 CalcNote 所使用的第三方服務處理個人資料有任何疑問或疑慮，歡迎直接與我們聯繫。請將您的詢問寄至 burton9999dev@gmail.com。我們致力於回應您的疑慮，並說明應用程式內使用者資料的管理方式。

本隱私權政策受日本法律管轄並依其解釋，不適用法律衝突原則。因使用 CalcNote 或與本隱私權政策相關而產生的任何爭議，均由日本境內法院享有專屬管轄權。使用 CalcNote 即表示使用者同意上述法院的管轄權與審理地。
