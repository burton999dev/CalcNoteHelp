## 1. Inledning
Denna integritetspolicy beskriver hur CalcNote, en kalkylatorapp, hanterar personuppgifter. Vi betonar att CalcNote i sig inte samlar in någon personlig information eller platsdata från sina användare. Det är dock viktigt att notera att tredjepartsbibliotek som används i appen kan samla in viss information. Policyn syftar till att på ett transparent sätt förklara hur data hanteras och för vilka ändamål den kan användas.

## 2. Insamling och användning av personuppgifter
CalcNote är utformad för att prioritera användarnas integritet. Därför samlar appen i sig inte in någon personlig information, såsom namn, e-postadresser eller platsdata. Trots detta använder CalcNote följande tredjepartsbibliotek, som kan samla in information:

- Admob: Används för att visa annonser. Insamlad data kan inkludera användarinteraktion med annonser och anonyma identifierare.
- Firebase Analytics: Hjälper till att förstå användarbeteende i appen genom att samla in anonym data om användning och interaktioner.
- Firebase Crashlytics: Används för att övervaka appens prestanda och identifiera krascher. Samlar in data om krascher och systeminformation, vilket är viktigt för att förbättra appens stabilitet.

Varje tjänst fungerar enligt sin egen integritetspolicy och kan behandla data i enlighet med sina riktlinjer. Användare uppmuntras att läsa de respektive integritetspolicyerna för mer detaljerad information. Det är viktigt att klargöra att även om dessa tjänster samlar in data för specifika ändamål såsom annonsering, analys och appprestanda, är CalcNote i sig ett verktyg som inte självständigt samlar in personuppgifter.

## 3. Databehandling av tredje part
CalcNote använder flera tredjepartstjänster för att förbättra användarupplevelsen och appens funktionalitet. Dessa tjänster, inklusive Admob, Firebase Analytics och Firebase Crashlytics, omfattas av sina egna integritetspolicyer och villkor för databehandling. De hanterar användardata i enlighet med dataskyddsförordningen (GDPR) och andra tillämpliga lagar.

- Admob: Hanterar annonsdata och säkerställer att relevanta och riktade annonser visas för användare. Mer information finns i [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html).
- Firebase Analytics och Firebase Crashlytics: Används för att förstå användarinteraktioner i appen och upprätthålla appens stabilitet. Detaljerad information finns i [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms).

Vi rekommenderar att användare granskar dessa dokument för att förstå hur deras data behandlas och skyddas av dessa tredjepartstjänster.

## 4. Användarens rättigheter
Enligt GDPR har användare specifika rättigheter gällande sina personuppgifter. Dessa inkluderar:

- Rätt till tillgång: Användare kan begära information om hur deras data används och behandlas.
- Rätt till rättelse: Användare har rätt att korrigera felaktiga eller ofullständiga uppgifter.
- Rätt till radering: Även kallad ”rätten att bli bortglömd”, vilket ger användare möjlighet att begära radering under vissa omständigheter.
- Rätt till begränsning av behandling: Användare kan begära att behandlingen av deras data stoppas.
- Rätt till dataportabilitet: Detta gör det möjligt för användare att få och återanvända sina data för egna syften i olika tjänster.
- Rätt att invända: Användare kan invända mot behandlingen av sina personuppgifter i vissa fall, till exempel för direktmarknadsföring.

CalcNote uppmuntrar användare att utöva dessa rättigheter vid behov. För förfrågningar eller frågor om personuppgifter kan användare kontakta den relevanta tredjepartsleverantören direkt, eftersom CalcNote inte själv behandlar personuppgifter.

## 5. Dataskydd
Säkerheten för användardata är en prioritet för CalcNote. Även om appen inte samlar in personuppgifter säkerställer vi att integrerade tredjepartstjänster följer strikta dataskyddsstandarder. Dessa tjänster måste implementera lämpliga säkerhetsåtgärder för att skydda användardata mot obehörig åtkomst, ändring, avslöjande eller förstöring. I det osannolika fallet av en dataincident kommer den berörda tredjepartstjänsten att vidta nödvändiga åtgärder för att minska eventuell skada och informera berörda användare enligt lag.

## 6. Ändringar i integritetspolicyn
CalcNote förbehåller sig rätten att när som helst uppdatera eller ändra denna integritetspolicy. Ändringar kommer att kommuniceras via appen eller den officiella webbplatsen, så att användare kan granska de uppdaterade villkoren. Den reviderade policyn träder i kraft från publiceringsdatumet. Vi uppmuntrar användare att regelbundet granska integritetspolicyn för att hålla sig informerade om hur deras data skyddas. Fortsatt användning av CalcNote efter ändringar innebär att användaren accepterar dessa ändringar.

## 8. Tillämplig lag och jurisdiktion
Denna integritetspolicy regleras av och tolkas enligt japansk lag, utan hänsyn till lagvalsprinciper. Eventuella tvister som uppstår ur eller i samband med användningen av CalcNote eller denna integritetspolicy ska omfattas av exklusiv jurisdiktion för domstolar belägna i Japan. Genom att använda CalcNote samtycker användare till jurisdiktionen och platsen för dessa domstolar.
