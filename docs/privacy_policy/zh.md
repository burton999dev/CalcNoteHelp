## 1. 引言
本隐私政策说明 CalcNote（一个计算器应用）在处理个人数据方面的做法。我们强调，CalcNote 本身不会收集用户的任何个人信息或位置信息。然而，需要注意的是，应用中使用的第三方库可能会收集某些信息。本政策旨在透明地说明数据如何被处理以及可能用于哪些目的。

## 2. 个人信息的收集与使用
CalcNote 的设计以用户隐私为优先。因此，应用本身不会收集任何个人信息，包括但不限于姓名、电子邮件地址或位置信息。不过，CalcNote 集成了以下第三方库，可能会收集信息：

- Admob：用于投放广告。收集的数据可能包括用户与广告的交互以及匿名标识符。
- Firebase Analytics：通过收集有关应用使用和交互的匿名数据，帮助了解用户在应用中的行为。
- Firebase Crashlytics：用于监控应用性能并识别崩溃。它会收集崩溃相关数据和系统信息，这对提升应用稳定性至关重要。

上述每项服务均依据其自身的隐私政策运行，并可能根据其指导方针处理数据。建议用户查阅相关服务的隐私政策以获取更多信息。需要明确的是，尽管这些服务会为了广告、分析和应用性能等特定目的收集数据，但 CalcNote 本身不会独立收集用户的个人数据。

## 3. 第三方数据处理
CalcNote 使用多项第三方服务以提升用户体验和应用功能。这些服务（包括 Admob、Firebase Analytics 和 Firebase Crashlytics）均受其各自的隐私政策与数据处理条款约束，并按照《通用数据保护条例》（GDPR）及其他适用法律处理用户数据。

- Admob：管理广告数据，确保向用户显示相关且有针对性的广告。详情请参阅 [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html)。
- Firebase Analytics 与 Firebase Crashlytics：用于理解应用内用户交互并维持应用稳定性。详细信息请参阅 [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms)。

我们建议用户查看这些文件，以了解第三方服务如何处理并保护其数据。

## 4. 用户权利
根据 GDPR，用户对其个人数据享有特定权利，包括：

- 访问权：用户可以请求了解其数据如何被使用和处理。
- 更正权：用户有权更正不准确或不完整的数据。
- 删除权：亦称“被遗忘权”，允许用户在特定情况下请求删除数据。
- 限制处理权：用户可以请求暂停处理其数据。
- 数据可携带权：允许用户获取并在不同服务之间为自身目的再利用其数据。
- 反对权：用户可在某些情况下（例如用于直接营销）反对处理其个人数据。

CalcNote 鼓励用户在需要时行使这些权利。由于 CalcNote 本身不处理个人数据，关于个人数据的请求或咨询，用户可直接联系相关第三方服务提供商。

## 5. 数据保护
用户数据的安全与保护是 CalcNote 的优先事项。虽然应用本身不收集个人数据，但我们确保集成的第三方服务遵循严格的数据保护标准。这些服务需实施适当的安全措施，以防止用户数据遭到未经授权的访问、篡改、披露或毁坏。若发生数据泄露，相关第三方服务将采取必要措施以减轻潜在损害，并按法律要求通知受影响的用户。

## 6. 隐私政策的变更
CalcNote 保留随时更新或修改本隐私政策的权利。变更将通过应用或官方网站进行通知，以便用户查看更新后的条款。修订后的政策自发布之日起生效。我们建议用户定期查看隐私政策，以了解其数据如何受到保护。在隐私政策变更后继续使用 CalcNote 即表示接受这些变更。

## 7. 联系方式
如对本隐私政策或 CalcNote 所使用的第三方服务处理个人数据有任何疑问或担忧，欢迎直接联系我们。请将您的咨询发送至 burton9999dev@gmail.com。我们致力于回应您的关切，并说明应用内用户数据的管理方式。

## 8. 适用法律与管辖
本隐私政策受日本法律管辖并按其解释，不适用法律冲突原则。因使用 CalcNote 或与本隐私政策相关而产生的任何争议，均由位于日本的法院专属管辖。使用 CalcNote 即表示用户同意上述法院的管辖权及审理地点。
