## 1. Introdução
Esta Política de Privacidade descreve a abordagem do CalcNote, um aplicativo de calculadora, no tratamento de dados pessoais. Enfatizamos que o CalcNote não coleta nenhuma informação pessoal ou dados de localização de seus usuários. No entanto, é importante observar que bibliotecas de terceiros usadas no aplicativo podem coletar algumas informações. Esta política visa explicar de forma transparente como os dados são tratados e para quais finalidades podem ser utilizados.

## 2. Coleta e uso de informações pessoais
O CalcNote foi projetado para priorizar a privacidade do usuário. Assim, o próprio aplicativo não coleta nenhuma informação pessoal, incluindo, entre outras, nomes, endereços de e-mail ou dados de localização. Contudo, o CalcNote incorpora as seguintes bibliotecas de terceiros, que podem coletar informações:

- Admob: Usado para veicular anúncios. Os dados coletados podem incluir a interação do usuário com anúncios e identificadores anônimos.
- Firebase Analytics: Ajuda a entender o comportamento do usuário no aplicativo por meio da coleta de dados anônimos sobre uso e interações.
- Firebase Crashlytics: Utilizado para monitorar o desempenho do aplicativo e identificar falhas. Coleta dados sobre falhas e informações do sistema, essenciais para melhorar a estabilidade do aplicativo.

Cada um desses serviços opera sob sua própria política de privacidade e pode processar dados de acordo com suas diretrizes. Os usuários são incentivados a revisar as políticas de privacidade desses serviços para mais detalhes. É importante esclarecer que, embora esses serviços coletem dados para finalidades específicas como publicidade, análise e desempenho do aplicativo, o CalcNote permanece como uma ferramenta que não coleta de forma independente dados pessoais dos usuários.

## 3. Tratamento de dados por terceiros
O CalcNote incorpora vários serviços de terceiros para aprimorar a experiência do usuário e a funcionalidade do aplicativo. Esses serviços, incluindo Admob, Firebase Analytics e Firebase Crashlytics, são regidos por suas respectivas políticas de privacidade e termos de processamento de dados. Eles tratam os dados dos usuários em conformidade com o Regulamento Geral de Proteção de Dados (GDPR) e outras leis aplicáveis.

- Admob: Gerencia dados de publicidade, garantindo que anúncios relevantes e direcionados sejam exibidos aos usuários. Para mais informações, consulte os [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html).
- Firebase Analytics e Firebase Crashlytics: Utilizados para entender as interações dos usuários no aplicativo e manter a estabilidade do app. Informações detalhadas podem ser encontradas nos [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms).

Recomendamos que os usuários consultem esses documentos para entender como seus dados são processados e protegidos por esses serviços de terceiros.

## 4. Direitos do usuário
De acordo com o GDPR, os usuários têm direitos específicos em relação aos seus dados pessoais. Eles incluem:

- Direito de acesso: Os usuários podem solicitar informações sobre como seus dados estão sendo usados e processados.
- Direito de retificação: Os usuários têm o direito de corrigir dados incorretos ou incompletos.
- Direito de exclusão: Também conhecido como “direito ao esquecimento”, permite solicitar a exclusão dos dados em determinadas circunstâncias.
- Direito de restringir o processamento: Os usuários podem solicitar a interrupção do processamento de seus dados.
- Direito à portabilidade dos dados: Permite obter e reutilizar seus dados para fins próprios em diferentes serviços.
- Direito de objeção: Os usuários podem se opor ao processamento de seus dados pessoais em determinados casos, como para fins de marketing direto.

O CalcNote incentiva os usuários a exercerem esses direitos quando necessário. Para solicitações ou dúvidas relacionadas a dados pessoais, os usuários podem entrar em contato diretamente com o provedor de serviços de terceiros relevante, já que o CalcNote não processa dados pessoais por conta própria.

## 5. Proteção de dados
A segurança dos dados dos usuários é uma prioridade para o CalcNote. Embora o aplicativo não colete dados pessoais, garantimos que os serviços de terceiros integrados sigam padrões rigorosos de proteção de dados. Esses serviços são obrigados a implementar medidas de segurança adequadas para proteger os dados dos usuários contra acesso não autorizado, alteração, divulgação ou destruição. No improvável caso de uma violação de dados, o serviço de terceiros responsável adotará as medidas necessárias para mitigar qualquer dano potencial e informará os usuários afetados conforme exigido por lei.

## 6. Alterações na Política de Privacidade
O CalcNote se reserva o direito de atualizar ou modificar esta Política de Privacidade a qualquer momento. As alterações serão comunicadas por meio do aplicativo ou do site oficial, permitindo que os usuários revisem os termos atualizados. A política revisada entrará em vigor a partir da data de publicação. Incentivamos os usuários a revisar periodicamente a Política de Privacidade para se manterem informados sobre como seus dados estão sendo protegidos. O uso continuado do CalcNote após quaisquer alterações na Política de Privacidade significa a aceitação dessas alterações.

## 7. Informações de contato
Para quaisquer dúvidas ou preocupações relacionadas a esta Política de Privacidade ou ao tratamento de dados pessoais pelos serviços de terceiros usados no CalcNote, os usuários são incentivados a entrar em contato conosco diretamente. Envie suas solicitações para burton9999dev@gmail.com. Estamos comprometidos em responder às suas preocupações e fornecer clareza sobre como os dados dos usuários são gerenciados no aplicativo.

## 8. Lei aplicável e jurisdição
Esta Política de Privacidade é regida e interpretada de acordo com as leis do Japão, sem dar efeito a quaisquer princípios de conflito de leis. Qualquer disputa decorrente do uso do CalcNote ou relacionada a esta Política de Privacidade estará sujeita à jurisdição exclusiva dos tribunais localizados no Japão. Ao usar o CalcNote, os usuários concordam com a jurisdição e o foro desses tribunais.
