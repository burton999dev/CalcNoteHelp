## 1. Pengenalan
Dasar Privasi ini menerangkan pendekatan CalcNote, aplikasi kalkulator, dalam mengendalikan data peribadi. Kami menekankan bahawa CalcNote sendiri tidak mengumpul sebarang maklumat peribadi atau data lokasi daripada penggunanya. Walau bagaimanapun, penting untuk diperhatikan bahawa perpustakaan pihak ketiga yang digunakan dalam aplikasi mungkin mengumpul maklumat tertentu. Dasar ini bertujuan untuk menerangkan secara telus bagaimana data dikendalikan dan tujuan penggunaannya.

## 2. Pengumpulan dan penggunaan maklumat peribadi
CalcNote direka untuk mengutamakan privasi pengguna. Oleh itu, aplikasi ini sendiri tidak mengumpul sebarang maklumat peribadi, termasuk tetapi tidak terhad kepada nama, alamat e-mel atau data lokasi. Namun, CalcNote menggabungkan perpustakaan pihak ketiga berikut yang mungkin mengumpul maklumat:

- Admob: Digunakan untuk memaparkan iklan. Data yang dikumpulkan mungkin termasuk interaksi pengguna dengan iklan dan pengecam tanpa nama.
- Firebase Analytics: Membantu memahami tingkah laku pengguna dalam aplikasi dengan mengumpul data tanpa nama mengenai penggunaan dan interaksi.
- Firebase Crashlytics: Digunakan untuk memantau prestasi aplikasi dan mengenal pasti kerosakan. Mengumpul data berkaitan kerosakan dan maklumat sistem, yang penting untuk meningkatkan kestabilan aplikasi.

Setiap perkhidmatan ini beroperasi di bawah dasar privasinya sendiri dan mungkin memproses data mengikut garis panduannya. Pengguna digalakkan untuk menyemak dasar privasi perkhidmatan tersebut bagi mendapatkan maklumat yang lebih terperinci. Adalah penting untuk menjelaskan bahawa walaupun perkhidmatan ini mengumpul data untuk tujuan khusus seperti pengiklanan, analitik dan prestasi aplikasi, CalcNote sendiri kekal sebagai alat yang tidak mengumpul data peribadi pengguna secara bebas.

## 3. Pemprosesan data oleh pihak ketiga
CalcNote menggabungkan beberapa perkhidmatan pihak ketiga untuk meningkatkan pengalaman pengguna dan fungsi aplikasi. Perkhidmatan ini, termasuk Admob, Firebase Analytics dan Firebase Crashlytics, ditadbir oleh dasar privasi dan terma pemprosesan data masing-masing. Mereka mengendalikan data pengguna selaras dengan General Data Protection Regulation (GDPR) dan undang-undang lain yang berkenaan.

- Admob: Mengurus data pengiklanan, memastikan iklan yang disasarkan dan relevan dipaparkan kepada pengguna. Untuk maklumat lanjut, sila rujuk [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html).
- Firebase Analytics dan Firebase Crashlytics: Digunakan untuk memahami interaksi pengguna dalam aplikasi dan mengekalkan kestabilan aplikasi. Maklumat terperinci boleh didapati dalam [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms).

Kami mengesyorkan pengguna untuk menyemak dokumen ini bagi memahami bagaimana data mereka diproses dan dilindungi oleh perkhidmatan pihak ketiga ini.

## 4. Hak pengguna
Di bawah GDPR, pengguna mempunyai hak tertentu berkaitan data peribadi mereka. Ini termasuk:

- Hak untuk akses: Pengguna boleh meminta maklumat tentang bagaimana data mereka digunakan dan diproses.
- Hak untuk pembetulan: Pengguna berhak membetulkan data yang tidak tepat atau tidak lengkap.
- Hak untuk pemadaman: Juga dikenali sebagai â€œhak untuk dilupakanâ€, membolehkan pengguna meminta pemadaman data dalam keadaan tertentu.
- Hak untuk mengehadkan pemprosesan: Pengguna boleh meminta penghentian pemprosesan data mereka.
- Hak kebolehpindahan data: Membolehkan pengguna memperoleh dan menggunakan semula data mereka untuk tujuan sendiri merentas perkhidmatan yang berbeza.
- Hak untuk membantah: Pengguna boleh membantah pemprosesan data peribadi mereka dalam keadaan tertentu, seperti untuk tujuan pemasaran langsung.

CalcNote menggalakkan pengguna untuk menggunakan hak-hak ini apabila perlu. Untuk permintaan atau pertanyaan berkaitan data peribadi, pengguna boleh menghubungi pembekal perkhidmatan pihak ketiga yang berkaitan secara terus, kerana CalcNote tidak memproses data peribadi sendiri.

## 5. Perlindungan data
Keselamatan dan perlindungan data pengguna adalah keutamaan bagi CalcNote. Walaupun aplikasi tidak mengumpul data peribadi, kami memastikan bahawa perkhidmatan pihak ketiga yang disepadukan mematuhi standard perlindungan data yang ketat. Perkhidmatan ini dikehendaki melaksanakan langkah keselamatan yang sesuai untuk melindungi data pengguna daripada akses tanpa kebenaran, perubahan, pendedahan atau pemusnahan. Dalam keadaan yang tidak dijangka berlaku pelanggaran data, perkhidmatan pihak ketiga yang berkenaan akan mengambil langkah yang diperlukan untuk mengurangkan sebarang potensi kerugian dan akan memaklumkan pengguna yang terjejas seperti yang dikehendaki oleh undang-undang.

## 6. Perubahan pada Dasar Privasi
CalcNote berhak untuk mengemas kini atau mengubah Dasar Privasi ini pada bila-bila masa. Perubahan akan dimaklumkan melalui aplikasi atau laman web rasmi, membolehkan pengguna menyemak terma yang dikemas kini. Dasar yang disemak berkuat kuasa dari tarikh penerbitan. Kami menggalakkan pengguna untuk menyemak Dasar Privasi secara berkala bagi kekal dimaklumkan tentang bagaimana data mereka dilindungi. Penggunaan berterusan CalcNote selepas sebarang perubahan pada Dasar Privasi menandakan penerimaan terhadap perubahan tersebut.
