## 1. Введение
Настоящая Политика конфиденциальности описывает подход CalcNote, приложения-калькулятора, к обработке персональных данных. Мы подчеркиваем, что CalcNote сам по себе не собирает личную информацию или данные о местоположении пользователей. Однако важно отметить, что сторонние библиотеки, используемые в приложении, могут собирать определенные сведения. Эта политика призвана прозрачно объяснить, как обрабатываются данные и для каких целей они могут использоваться.

## 2. Сбор и использование персональной информации
CalcNote разработан с приоритетом на конфиденциальность пользователей. Поэтому само приложение не собирает персональную информацию, включая, помимо прочего, имена, адреса электронной почты или данные о местоположении. Тем не менее CalcNote использует следующие сторонние библиотеки, которые могут собирать информацию:

- Admob: Используется для показа рекламы. Собираемые данные могут включать взаимодействие пользователя с рекламой и анонимные идентификаторы.
- Firebase Analytics: Помогает понимать поведение пользователей в приложении, собирая анонимные данные об использовании и взаимодействиях.
- Firebase Crashlytics: Используется для мониторинга производительности приложения и выявления сбоев. Собирает данные о сбоях и системную информацию, важные для улучшения стабильности приложения.

Каждый из этих сервисов работает в соответствии со своей политикой конфиденциальности и может обрабатывать данные согласно своим правилам. Пользователям рекомендуется ознакомиться с соответствующими политиками конфиденциальности для получения более подробной информации. Важно отметить, что хотя эти сервисы собирают данные для конкретных целей, таких как реклама, аналитика и производительность приложения, CalcNote сам по себе не собирает персональные данные пользователей.

## 3. Обработка данных третьими лицами
CalcNote использует несколько сторонних сервисов для улучшения пользовательского опыта и функциональности приложения. Эти сервисы, включая Admob, Firebase Analytics и Firebase Crashlytics, регулируются собственными политиками конфиденциальности и условиями обработки данных. Они обрабатывают пользовательские данные в соответствии с Общим регламентом по защите данных (GDPR) и другими применимыми законами.

- Admob: Управляет рекламными данными, обеспечивая показ целевых и релевантных объявлений пользователям. Подробнее см. в [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html).
- Firebase Analytics и Firebase Crashlytics: Используются для понимания взаимодействий пользователей в приложении и поддержания стабильности приложения. Подробная информация доступна в [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms).

Мы рекомендуем пользователям ознакомиться с этими документами, чтобы понять, как их данные обрабатываются и защищаются этими сторонними сервисами.

## 4. Права пользователей
Согласно GDPR пользователи имеют определенные права в отношении своих персональных данных. К ним относятся:

- Право на доступ: Пользователи могут запросить информацию о том, как используются и обрабатываются их данные.
- Право на исправление: Пользователи имеют право исправлять неточные или неполные данные.
- Право на удаление: Также известно как «право быть забытым»; позволяет запросить удаление данных при определенных обстоятельствах.
- Право на ограничение обработки: Пользователи могут запросить приостановку обработки своих данных.
- Право на переносимость данных: Позволяет пользователям получать и повторно использовать свои данные для собственных целей в разных сервисах.
- Право на возражение: Пользователи могут возражать против обработки своих персональных данных в определенных случаях, например для целей прямого маркетинга.

CalcNote поощряет пользователей пользоваться этими правами при необходимости. По вопросам или запросам, связанным с персональными данными, пользователи могут обращаться напрямую к соответствующему стороннему поставщику услуг, поскольку CalcNote сам не обрабатывает персональные данные.

## 5. Защита данных
Безопасность и защита пользовательских данных являются приоритетом для CalcNote. Хотя приложение не собирает персональные данные, мы обеспечиваем соблюдение строгих стандартов защиты данных интегрированными сторонними сервисами. Эти сервисы обязаны внедрять соответствующие меры безопасности для защиты пользовательских данных от несанкционированного доступа, изменения, раскрытия или уничтожения. В маловероятном случае утечки данных соответствующий сторонний сервис предпримет необходимые шаги для минимизации возможного ущерба и уведомит затронутых пользователей в соответствии с требованиями закона.
