## 1. はじめに
このプライバシーポリシーは、電卓アプリ CalcNote における個人データの取り扱い方針を示します。CalcNote 自体はユーザーの個人情報や位置情報を収集しないことを強調します。ただし、アプリ内で使用しているサードパーティ製ライブラリが一定の情報を収集する場合があります。本ポリシーは、データの取り扱いと利用目的を透明性をもって説明することを目的としています。

## 2. 個人情報の収集と利用
CalcNote はユーザーのプライバシーを最優先に設計されています。そのため、アプリ自体は氏名、メールアドレス、位置情報などの個人情報を一切収集しません。しかし、CalcNote には以下のサードパーティライブラリが組み込まれており、これらが情報を収集する場合があります。

- Admob: 広告配信に使用されます。収集されるデータには、広告への操作や匿名識別子が含まれる場合があります。
- Firebase Analytics: アプリ内の利用状況や操作に関する匿名データを収集し、ユーザー行動の理解に役立てます。
- Firebase Crashlytics: アプリのパフォーマンス監視とクラッシュの特定に利用されます。アプリの安定性向上に重要なクラッシュ情報やシステム情報を収集します。

これらのサービスはそれぞれ独自のプライバシーポリシーに基づいて運用され、そのガイドラインに従ってデータを処理する場合があります。詳細については、各サービスのプライバシーポリシーをご確認ください。これらのサービスは広告、解析、アプリの安定性といった目的でデータを収集しますが、CalcNote 自体は個人情報を独自に収集しないツールであることを明確にします。

## 3. 第三者によるデータ処理
CalcNote はユーザー体験とアプリ機能を向上させるために複数のサードパーティサービスを利用しています。Admob、Firebase Analytics、Firebase Crashlytics などのサービスは、それぞれのプライバシーポリシーおよびデータ処理条件に従って運用されます。これらのサービスは、GDPR（一般データ保護規則）やその他適用される法律に準拠してユーザーデータを取り扱います。

- Admob: 広告データを管理し、ユーザーに関連性の高い広告を表示します。詳細は [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html) をご覧ください。
- Firebase Analytics および Firebase Crashlytics: アプリ内のユーザー行動の理解とアプリの安定性維持に利用されます。詳細は [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms) をご覧ください。

これらの文書を確認することで、第三者サービスがどのようにデータを処理し、保護しているかを理解できます。

## 4. ユーザーの権利
GDPR の下で、ユーザーは個人データに関して次の権利を有します。

- アクセス権: データの使用・処理方法についての情報を請求できます。
- 訂正権: 誤りや不完全なデータの修正を求める権利があります。
- 消去権: 「忘れられる権利」とも呼ばれ、一定の条件下でデータの削除を求めることができます。
- 処理制限権: データ処理の停止を求めることができます。
- データポータビリティ権: 自身のデータを取得し、他サービスで再利用することができます。
- 異議申立て権: 直接マーケティング目的など、特定のケースにおけるデータ処理に異議を申し立てることができます。

CalcNote は必要に応じてこれらの権利の行使を促します。CalcNote は個人データを自ら処理しないため、個人データに関する要望やお問い合わせは該当する第三者サービス提供者へ直接ご連絡ください。

## 5. データ保護
ユーザーデータの安全性とセキュリティは CalcNote にとって最優先事項です。アプリ自体は個人データを収集しませんが、統合された第三者サービスが厳格なデータ保護基準に従うことを確保しています。これらのサービスは、不正アクセス、改ざん、開示、破壊からユーザーデータを守るための適切なセキュリティ対策を実施することが求められています。万一データ侵害が発生した場合、当該第三者サービスは必要な措置を講じ、法令に基づき影響を受けるユーザーへ通知します。

## 6. プライバシーポリシーの変更
CalcNote はいつでも本プライバシーポリシーを更新または変更する権利を有します。変更はアプリまたは公式サイトで告知し、ユーザーが更新内容を確認できるようにします。改定版は公開日から効力を生じます。ユーザーには定期的な確認を推奨します。変更後に CalcNote を継続利用した場合、変更に同意したものとみなされます。

## 8. 準拠法および管轄
本プライバシーポリシーは抵触法の原則にかかわらず、日本法に準拠し、これに従って解釈されます。CalcNote の使用または本プライバシーポリシーに関連して生じる一切の紛争は、日本の裁判所の専属的管轄に服します。CalcNote を利用することで、ユーザーは当該裁判所の管轄および裁判地に同意したものとします。
