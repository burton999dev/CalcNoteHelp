## 1. บทนำ
นโยบายความเป็นส่วนตัวฉบับนี้อธิบายแนวทางของ CalcNote ซึ่งเป็นแอปเครื่องคิดเลขในการจัดการข้อมูลส่วนบุคคล เราเน้นย้ำว่า CalcNote เองไม่ได้เก็บรวบรวมข้อมูลส่วนบุคคลหรือข้อมูลตำแหน่งที่ตั้งของผู้ใช้ อย่างไรก็ตาม ควรทราบว่าไลบรารีของบุคคลที่สามที่ใช้ในแอปอาจเก็บรวบรวมข้อมูลบางอย่างได้ นโยบายนี้มีวัตถุประสงค์เพื่ออธิบายอย่างโปร่งใสว่ามีการจัดการข้อมูลอย่างไรและเพื่อวัตถุประสงค์ใด

## 2. การเก็บรวบรวมและการใช้ข้อมูลส่วนบุคคล
CalcNote ถูกออกแบบมาให้ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้ ดังนั้นตัวแอปเองจึงไม่เก็บรวบรวมข้อมูลส่วนบุคคล เช่น ชื่อ ที่อยู่อีเมล หรือข้อมูลตำแหน่งที่ตั้ง อย่างไรก็ตาม CalcNote มีการผสานไลบรารีของบุคคลที่สามต่อไปนี้ซึ่งอาจเก็บรวบรวมข้อมูล:

- Admob: ใช้สำหรับแสดงโฆษณา ข้อมูลที่เก็บรวบรวมอาจรวมถึงการโต้ตอบของผู้ใช้กับโฆษณาและตัวระบุแบบไม่ระบุตัวตน
- Firebase Analytics: ช่วยทำความเข้าใจพฤติกรรมผู้ใช้ภายในแอปโดยการเก็บรวบรวมข้อมูลแบบไม่ระบุตัวตนเกี่ยวกับการใช้งานและการโต้ตอบ
- Firebase Crashlytics: ใช้สำหรับเฝ้าระวังประสิทธิภาพของแอปและระบุการล้มเหลว โดยเก็บรวบรวมข้อมูลการล้มเหลวและข้อมูลระบบ ซึ่งมีความสำคัญต่อการปรับปรุงเสถียรภาพของแอป

บริการแต่ละรายการดำเนินการภายใต้นโยบายความเป็นส่วนตัวของตนเอง และอาจประมวลผลข้อมูลตามแนวทางของตน ผู้ใช้ควรตรวจสอบนโยบายความเป็นส่วนตัวของแต่ละบริการเพื่อรับข้อมูลเพิ่มเติม ทั้งนี้ควรชี้แจงว่าแม้บริการเหล่านี้จะเก็บข้อมูลเพื่อวัตถุประสงค์เฉพาะ เช่น โฆษณา การวิเคราะห์ และประสิทธิภาพของแอป แต่ CalcNote เองยังคงเป็นเครื่องมือที่ไม่ได้เก็บข้อมูลส่วนบุคคลของผู้ใช้ด้วยตนเอง

## 3. การประมวลผลข้อมูลโดยบุคคลที่สาม
CalcNote ใช้บริการของบุคคลที่สามหลายรายการเพื่อยกระดับประสบการณ์ผู้ใช้และฟังก์ชันการทำงานของแอป บริการเหล่านี้ ได้แก่ Admob, Firebase Analytics และ Firebase Crashlytics ซึ่งอยู่ภายใต้นโยบายความเป็นส่วนตัวและเงื่อนไขการประมวลผลข้อมูลของตนเอง โดยจะจัดการข้อมูลผู้ใช้ตามข้อกำหนดของ GDPR และกฎหมายอื่นที่เกี่ยวข้อง

- Admob: จัดการข้อมูลโฆษณาเพื่อให้แสดงโฆษณาที่ตรงเป้าหมายและเกี่ยวข้องแก่ผู้ใช้ สำหรับรายละเอียดเพิ่มเติม โปรดดู [Google Ads Data Processing Terms](https://www.google.com/analytics/terms/dpa/dataprocessingamendment_20200816.html)
- Firebase Analytics และ Firebase Crashlytics: ใช้เพื่อทำความเข้าใจการโต้ตอบของผู้ใช้ในแอปและรักษาเสถียรภาพของแอป รายละเอียดเพิ่มเติมอยู่ใน [Firebase Data Processing Terms](https://firebase.google.com/terms/data-processing-terms)

เราแนะนำให้ผู้ใช้ตรวจสอบเอกสารเหล่านี้เพื่อทำความเข้าใจว่าข้อมูลของตนถูกประมวลผลและได้รับการคุ้มครองโดยบริการของบุคคลที่สามเหล่านี้อย่างไร

## 4. สิทธิของผู้ใช้
ภายใต้ GDPR ผู้ใช้มีสิทธิบางประการเกี่ยวกับข้อมูลส่วนบุคคลของตน ซึ่งรวมถึง:

- สิทธิในการเข้าถึง: ผู้ใช้สามารถขอข้อมูลเกี่ยวกับการใช้และการประมวลผลข้อมูลของตนได้
- สิทธิในการแก้ไข: ผู้ใช้มีสิทธิแก้ไขข้อมูลที่ไม่ถูกต้องหรือไม่ครบถ้วน
- สิทธิในการลบ: หรือ “สิทธิที่จะถูกลืม” ซึ่งอนุญาตให้ผู้ใช้ขอให้ลบข้อมูลในบางกรณี
- สิทธิในการจำกัดการประมวลผล: ผู้ใช้สามารถขอให้หยุดการประมวลผลข้อมูลของตนได้
- สิทธิในการพกพาข้อมูล: อนุญาตให้ผู้ใช้ได้รับและนำข้อมูลของตนไปใช้ซ้ำเพื่อวัตถุประสงค์ของตนในบริการต่าง ๆ
- สิทธิในการคัดค้าน: ผู้ใช้สามารถคัดค้านการประมวลผลข้อมูลส่วนบุคคลในบางกรณี เช่น เพื่อการตลาดทางตรง

CalcNote สนับสนุนให้ผู้ใช้ใช้สิทธิเหล่านี้เมื่อจำเป็น สำหรับคำขอหรือข้อสงสัยเกี่ยวกับข้อมูลส่วนบุคคล ผู้ใช้สามารถติดต่อผู้ให้บริการบุคคลที่สามที่เกี่ยวข้องได้โดยตรง เนื่องจาก CalcNote ไม่ได้ประมวลผลข้อมูลส่วนบุคคลด้วยตนเอง

## 5. การคุ้มครองข้อมูล
ความปลอดภัยและความมั่นคงของข้อมูลผู้ใช้เป็นสิ่งสำคัญสำหรับ CalcNote แม้ว่าแอปจะไม่เก็บรวบรวมข้อมูลส่วนบุคคล แต่เรารับรองว่าบริการของบุคคลที่สามที่ผสานอยู่ปฏิบัติตามมาตรฐานการคุ้มครองข้อมูลอย่างเข้มงวด บริการเหล่านี้ต้องดำเนินมาตรการด้านความปลอดภัยที่เหมาะสมเพื่อปกป้องข้อมูลผู้ใช้จากการเข้าถึงโดยไม่ได้รับอนุญาต การเปลี่ยนแปลง การเปิดเผย หรือการทำลาย ในกรณีที่ไม่น่าจะเกิดการละเมิดข้อมูล บริการของบุคคลที่สามที่เกี่ยวข้องจะดำเนินการที่จำเป็นเพื่อลดความเสียหายที่อาจเกิดขึ้น และจะแจ้งผู้ใช้ที่ได้รับผลกระทบตามที่กฎหมายกำหนด

## 6. การเปลี่ยนแปลงนโยบายความเป็นส่วนตัว
